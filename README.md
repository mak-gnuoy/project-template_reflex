# Python Project Template for Reflex

This is a python project tempate for vscode using devcontainer.

## Directory Layout
```
├── project-template_python-reflex
    ├── Dockerfile
    ├── LICENSE
    ├── README.md
    ├── app/
    |-- assets/
    ├── conf/
    ├── docs/
    ├── requirements.txt
    ├── src/
    |── tests/
    └── .web/
    
```

## Run
```
$ reflex run --loglevel debug
```
